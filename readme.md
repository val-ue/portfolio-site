Portfolio Site

OVERVIEW
This is a personal prtfolio site I created to showcase my projects.

FEATURES
Responsive Pages: Optimized for both desktop and mobile viewing.
Interactive Links: JavaScript was used to change the color 
of links on hover to enhance interactivity.
Project Grid: Showcases projects with descriptions, images, and links.


To run this project, follow these steps:
1.Clone the repository
2.Navigate to the project directory: portfolio-site
3.Open the index.html file in your browser to view the site.
